    PostGIS is a spatial database extender for PostgreSQL object-relational database. It adds support for geographic objects allowing location queries to be run in SQL.
    PL/Java is a free implementation of Java stored procedures, triggers, and functions for PostgreSQL. It allows you to write database functions in Java, and have them executed in the database as though they were implemented in SQL.
